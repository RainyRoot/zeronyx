What's inside

Everything a pentester needs

ZeroNyx orchestrates your tools, parses their output, and surfaces what matters — so you can focus on the engagement.

🔍

Unified Scan Interface

Run Nmap, Nuclei, Nikto, Gobuster, SQLMap, and more from a single interface. Real-time output streaming over WebSocket.

🌐

HTTP(S) Proxy

Built-in mitmproxy integration with request/response inspection, modification, and replay. Full history with search.

🤖

AI Analysis

Run findings through Ollama, OpenAI, or Anthropic. Prioritized risk assessment, false-positive detection, and next-step recommendations.

⛓️

Chain Automation

Define multi-step workflows that auto-run on scan completion. Quick Recon, Full Web Audit, Network Sweep — all automated.

📊

Finding Management

Normalized findings across all tools. CVSS scoring, CVE links, evidence attachments, and status tracking.

📝

Report Generation

Professional HTML/PDF reports with AI-generated executive summaries and technical findings, ready for client delivery.

🔌

Plugin System

Extend ZeroNyx with community plugins. Full SDK for building custom backend integrations and frontend UI slots.

🔐

Credential Store

Captured credentials are stored securely and can be fed directly into follow-up Hydra brute-force attacks.

📦

Obsidian Sync

Automatically export scan notes as structured Markdown into your Obsidian vault after every scan completes.
